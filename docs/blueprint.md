# **App Name**: StitchPlan

## Core Features:

- Gantt Chart: Display processes on a drag-and-drop Gantt chart with days as column headers and machine names as row headers.
- Process Scheduling: Allow users to schedule processes by dragging and dropping tasks onto the Gantt chart.
- Duration Calculation: Calculate the duration of each task based on the SAM (Standard Allowed Minutes) and the order quantity.

## Style Guidelines:

- Primary color: Navy blue (#2E3A87) to convey reliability and precision.
- Background color: Light gray (#F0F4F8) to provide a clean and unobtrusive backdrop.
- Accent color: Teal (#3EB489) to highlight important interactive elements.
- Font: 'Inter' sans-serif for a modern, neutral, and readable interface.
- Use clear, minimalistic icons to represent different process types and machine states.
- Maintain a clean and organized layout with sufficient whitespace to ensure ease of use and quick comprehension.
- Use subtle transitions and animations to provide feedback on user interactions, like dragging and dropping tasks.
